# ClearMap-Viz
Satellite repo of ClearMap with package and scripts for visualization, annotation

## Installation
```bash
git clone git@github.com:edoumazane/clearmap-viz.git
cd clearmap-viz
conda env create -f environment.yml
conda activate clearmap-viz
pip install -e .
```
